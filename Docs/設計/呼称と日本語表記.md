# 呼称と日本語表記

更新日：2026年7月21日

## 基本方針

プレイヤーが人物関係とゲーム情報をすぐ理解できるよう、システムメッセージ、UI、データ、説明文の呼称は日本式にします。そのうえで、人物の会話、称号、儀礼などの要所に中国式の呼び方を時折混ぜ、世界の雰囲気を出します。

## 適用範囲

### 日本式で統一するもの

- システムメッセージ
- 人物、ユニット、流派などのデータ項目
- 人物関係の表示
- 能力、状態、任務、勝利条件の説明
- チュートリアルと操作案内

### 中国式の呼び方を使えるもの

- 登場人物同士の会話
- 流派内の親しみや上下関係を表す呼びかけ
- 儀礼、口上、異名、称号
- 中国風の響きを印象付けたい演出

会話の中でも常に中国式にする必要はなく、その人物の性格や場面に合うときだけ使います。

## 家族の呼び方

実際の家族関係は、原則として日本語で表します。

- 兄、兄さん
- 姉、姉さん
- 弟
- 妹

人物の口調や出身地を表現したい場面に限り、「哥哥」「姐姐」などを愛称として使う余地はありますが、通常の関係表示には使いません。

## 流派の呼び方

ゲーム上の関係表示や説明文では、意味が直ちに分かる日本語を使います。

- 先輩の男性門弟：兄弟子
- 先輩の女性門弟：姉弟子
- 後輩の男性門弟：弟弟子
- 後輩の女性門弟：妹弟子

会話や称号では、次の中国風呼称を時折使います。

- 師兄：先輩の男性門弟
- 師姐：先輩の女性門弟
- 師弟：後輩の男性門弟
- 師妹：後輩の女性門弟
- 大師兄：流派を代表する最古参級の男性門弟
- 大師姐：流派を代表する最古参級の女性門弟

## 表示例

人物情報では日本語を優先します。

> 洪鉄山　紅熊亭の兄弟子

会話では中国風呼称を味付けとして使えます。

> 「師兄！　悬铃关が破られました！」

初登場時やチュートリアルでは、必要に応じて意味を会話内で自然に補います。

> 「桃蓮師姐――私の姉弟子にあたる人だ」

## 使いすぎを避ける

- 中国語を知らないと任務や能力を理解できない文章にはしない
- 一つの会話へ馴染みのない呼称を大量に入れない
- UI上の役割名、勝利条件、操作説明は日本語で統一する
- 中国風呼称は、人物関係、感情、儀礼、流派らしさを見せる場面に使う
- 読み方を正式採用する段階で、カナ表記や用語集の要否を決める
