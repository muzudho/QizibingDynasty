# 看板割りと流派の敗北

更新日：2026年7月21日

## 基本方針

道場の門に掛けられた木の看板を、流派が公に存在している証とします。流派の大ボスが正式な勝負に敗れると、勝者は看板を割ります。この「看板割り」によって拠点制圧が成立します。

建物や村を焼き払うのではなく、看板、名誉、門弟の所属が移り変わることで天下統一を表現します。

## 看板の状態

- **掲示中**：流派が独立して活動している
- **挑戦中**：大ボスとの勝負が進行している
- **破損**：大ボスが敗れ、流派としての権威を失った
- **掛け替え**：勝者の支部または傘下道場になった
- **無看板**：どの流派にも属さない道場、廃道場または在野の集まり

看板には流派名、動物の印、色が描かれます。マップ上の建物を見ただけで、現在の支配流派と敗北の痕跡が分かるようにします。

## 看板割りの流れ

1. 挑戦者が道場の門前で名乗る
2. 防衛側の大ボスまたは代理の猛者と勝負する
3. 敗者が結果を認める
4. 勝者が木の看板を割る
5. 門弟それぞれが今後の所属を選ぶ
6. 新しい看板を掛けるか、無看板のまま残す
7. 勝敗の噂が旅人と街道を通じて周辺へ広がる

看板を割る行為は殺害より軽い一方、流派にとっては非常に重い儀式です。卑怯な手段で割った看板は正当と認められず、住民の反発や再戦を招きます。

## 敗れた大ボス

大ボスの敗北後の行動は、性格、勝負の内容、勝者への感情、残った門弟によって変わります。

- 勝者の門下に入り、元大ボスの兄弟子または客将になる
- 一部の門弟を連れて村を去り、放浪する
- 一人で在野に戻り、別の土地で再起を図る
- 飯店の主人など、戦わない元の仕事へ戻る
- 敗者を集める別組織へ加入する
- 平和期まで身を引き、次の八年周期に再登場する

大ボスは倒した時点で消滅する駒ではなく、その後も人物関係と世界の変化を生む存在にします。

## 敗れた門弟

門弟は流派全体で一括処理せず、少なくとも性格または集団ごとに進路が分かれます。

- **帰順**：勝者の流派へ入る
- **随行**：元大ボスについて村を去る
- **在野**：所属を捨て、その土地や街道に残る
- **帰郷**：戦いをやめて家族や本業へ戻る
- **落伍者組織へ加入**：敗者同士の新しい居場所へ向かう
- **残党化**：敗北を認めず、遠隔地の支部や山中で旧看板を掲げ続ける

帰順しやすさには、忠誠、野心、地元への愛着、勝者の評判、看板割りの正当性などを反映できます。

## 敗者の受け皿「灰猿団」案

**灰猿団（はいえんだん）**は、看板を失った大ボス、門弟、用心棒、荷運び人などが流れ着く無看板の集団です。仮称であり、正式採用は未決定です。

- 灰色は、元の流派色が混ざって消えたことを表す
- 猿は、どの山にも住み、拾った物で器用に生きる姿を表す
- 固定した本拠地を持たず、放棄された道場や荷車を転々とする
- 元流派の技が混在しており、編成と能力を予測しにくい
- 敗者を笑わず受け入れる一方、看板と王位には執着しない
- 傭兵、道直し、荷運び、盗賊のどれになるかは土地によって異なる

灰猿団が大きくなるほど、天下統一からこぼれ落ちた者が可視化されます。プレイヤーが敗者を乱暴に扱うと灰猿団が強くなり、丁寧に帰順させると小さくなる仕組みにできます。

既存候補の「金猿酒家」と動物が重なるため、両者を因縁のある組織にするか、どちらかの動物名を変更するかは今後決めます。

## 勝敗の情報は街道を通って伝わる

流派の敗北は、発生した瞬間に世界中へ共有されません。村、宿場、支店、山寨は、それぞれが最後に受け取った情報を信じます。

- 近隣の村には徒歩で数日後に伝わる
- 街道と宿場が整っていれば速く広がる
- 崖崩れ、増水、冬の峠によって噂が止まる
- 商人、占い師、伝令、逃げた門弟が情報を運ぶ
- 勝者は正式な使者を送り、看板の破片を勝利の証拠にできる
- 敗者側は敗北を隠したり、偽の勝利を触れ回ったりできる
- 山奥の支部では、すでに本店が敗れたことを知らず旧流派を名乗り続ける

住民が「ああ、そんな流派があったな」と過去形で語り始めるまでには時間差があります。看板を割ることが物理的な制圧、噂が届くことが社会的な制圧です。

## ゲーム上の情報段階

各土地が持つ流派情報を、次の段階で表現できます。

- **未確認**：噂すら届いていない
- **風聞**：敗北したらしいが、内容は不明
- **確認済み**：使者または信頼できる旅人から情報を得た
- **公認**：新しい看板と所属を土地が受け入れた

プレイヤー自身も遠方の敵について常に正確な情報を得られるとは限りません。地図には「紅熊亭支部？」のように、最後に確認した看板を表示できます。

## 未決定事項

- 看板割りを必ず大ボス戦後の演出にするか
- 看板の破片を戦利品、使者の証明、再戦の印として使うか
- 門弟の進路を個人単位で決めるか、集団単位で決めるか
- 遠隔支部が自動降伏するか、別流派として独立するか
- 灰猿団を正式採用するか
- 嘘の噂や情報遅延を、どの程度プレイヤー操作へ影響させるか
