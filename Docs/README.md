## フォルダーの用途

- `企画`：アイデア、原案、検討中のゲーム内容
- `設計`：仕様として固まったゲームシステムや画面設計
- `開発`：実装計画、調査記録、動作確認や復旧の記録
- `調査`：まだ結論が出ていない調査
- `運用`：リポジトリや資料を扱うためのルール
