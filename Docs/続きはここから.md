# 続きはここから

更新日：2026年7月24日

## 現在地点

新規ストラテジーゲーム「Qizibing Dynasty（棋子餅王朝）」の初期構想を整理しました。

決まっている内容は [企画/ゲーム構想.md](企画/ゲーム構想.md) にまとめています。フルHD基準の2Dクォータービュー、古代中国風の世界、流派と道場を持つプレイヤー、個性的なユニットの収集、各地の猛者を倒して全拠点を制圧するという骨格があります。

世界地理は海よりも山岳と大河を中心にし、天然の要塞である山中に拠点を点在させます。長い土の道や崖道で拠点を結び、経路の確保そのものを戦略にする方向です。また、中国風の親しみやすさは持たせつつ、特定の既存作品に似せないことを創作方針とします。

タイトルの「棋子餅」は菓子の名前から採ったもので、作品全体の共通項か主人公側の名前かは未決定です。「王朝」にも今のところ歴史的な意味はありません。どちらも後から世界観を育てるための種として、複数案を企画書に残しています。

実在の中国を舞台にせず、中国語の一般名詞を組み合わせた [企画/地名案.md](企画/地名案.md) を作成しました。最初の地方候補は、霧と棚田の山地「雾阶盆地」です。村から山道、崖の桟道、関所、山寨、大河の渡しを経て城郭都市へ至る十二の地名候補があります。すべて未確定です。

流派は「色＋動物＋飯店などの店種」で命名し、大ボス、幹部、一般兵にも共通の色や動物を持たせる案があります。天下に存在する大小の勢力の総称は「諸味百家」、プレイヤーが選択する勢力単位は「一家」とします。「看板」は一家の名称と独立、「流派」は受け継がれる料理や武術の体系を指します。[企画/流派と組織名案.md](企画/流派と組織名案.md) に、用語整理と紅熊亭、碧蛇茶楼、白鶴飯荘、黒牛麺館、金猿酒家、紫燕餃子房の組織案をまとめました。

前回の四年戦争の勝者は「黄虎大飯店」とする案が有力です。黄虎大飯店は平和期に複数の看板へ分かれて衰退しましたが、本店と前王者の信用を残しており、今回も連覇できます。一家には本家から分かれた「分看板」と、自分たちで掲げた「起こし看板」があります。また、弱小一家を単なる雑魚ではなく、選択可能で成長できる勢力とする案を [企画/弱小流派と寄合案.md](企画/弱小流派と寄合案.md) にまとめました。翠熊猫竹館、通称「大熊猫一家」を含む弱小一家61組と寄合6組を挙げ、動物と料理を血生臭くせず、コミカルで親しみやすく描く方針も記載しています。

一局の舞台内時間は約四年間とします。四年に一度、龍の姿に見える巨大な気象現象「天巡龍」が現れ、次の王を選ぶという伝承があります。プレイヤーたちはその出現までの全拠点制圧を目指します。四年×四季の進行案と、龍の正体を断定しない方針は [企画/四年周期と天の龍.md](企画/四年周期と天の龍.md) にあります。

世界では四年の戦争と四年の平和が交互に続くと信じられています。占い師が天巡龍を読み、その言葉を信じた民衆と流派が予言どおりに行動するため、周期が自己成就している可能性があります。科学水準は気象を説明できない一方、季節や風を読む経験知は豊富という方向です。

システムメッセージ、UI、データ、説明文の呼称は日本式にし、人物会話では中国式の呼び方を時折混ぜる方針を決定しました。データでは「兄弟子」「姉弟子」などを使い、会話や称号では「師兄」「師姐」「大師兄」などを味付けとして使います。詳細は [設計/呼称と日本語表記.md](設計/呼称と日本語表記.md) にあります。

流派の拠点制圧は、大ボスを倒して道場の木の看板を割る「看板割り」で表現します。敗れた大ボスと門弟は帰順、随行、在野、帰郷、残党化などへ分かれます。勝敗は世界へ即時共有されず、街道を行く旅人や使者によって徐々に広がります。敗者の受け皿となる無看板組織「灰猿団」も候補です。詳細は [設計/看板割りと流派の敗北.md](設計/看板割りと流派の敗北.md) にあります。

時間表現は、七日一週や五行属性をそのまま採用せず、独自暦にする案を作成しました。一年十二か月、一月三十日、十日ごとの旬、月に属さない五日間「空日」、四年目だけ加わる「龍余日」が中心です。システムでは日本式の日付、会話では「河月」などの別名を使います。詳細は [企画/暦と時間表現案.md](企画/暦と時間表現案.md) にあります。すべて正式採用前の候補です。

劇中世界にはアルファベットを登場させない方針を決定しました。英字の `Qizibing Dynasty` はタイトルロゴや製品情報などゲーム外だけに使い、ゲーム開始後の看板、地図、文書、小道具には漢字風の表意文字を使います。実在の地球史や国家が同じ世界に存在することも前提にしません。

ゲームの見た目を検討するフルHD比率の静止画モックアップを7枚作成しました。[企画/画面モックアップ](企画/画面モックアップ) に、タイトル画面、旧流派選択画面、戦闘画面、マップ画面、道場画面、拠点制圧後の画面、新しい一家選択画面があります。AI生成によるイメージボードであり、ゲームへの組み込みはまだ行っていません。

ゲーム開始時に選ぶ単位は代表者個人ではなく「一家」とします。代表者は一家の顔として大きく表示しますが、捕縛や隠居で交代してもプレイを続けられます。一家選択画面は、同じ大きさのカードを並べず、約九画面分の横長な連続絵巻とします。強い一家は必ず大きく見せますが、弱いのに巨大な看板、飾り、行列、身体、態度によって場所を取る一家も混ぜます。人物、旗、鍋、動物は画面端で見切れ、スクロールすると全体が現れます。一人用と多人数ターン制の選択手順を含む詳細は [設計/一家選択画面.md](設計/一家選択画面.md) にまとめました。新構成の一場面は [企画/画面モックアップ/07-一家選択画面-連続絵巻.png](企画/画面モックアップ/07-一家選択画面-連続絵巻.png) です。旧流派選択モックアップも配色と意匠の参考として残します。
