# 続きはここから

更新日：2026年7月21日

## 現在地点

新規ストラテジーゲーム「Qizibing Dynasty（棋子餅王朝）」の初期構想を整理しました。

決まっている内容は [企画/ゲーム構想.md](企画/ゲーム構想.md) にまとめています。フルHD基準の2Dクォータービュー、古代中国風の世界、流派と道場を持つプレイヤー、個性的なユニットの収集、各地の猛者を倒して全拠点を制圧するという骨格があります。

世界地理は海よりも山岳と大河を中心にし、天然の要塞である山中に拠点を点在させます。長い土の道や崖道で拠点を結び、経路の確保そのものを戦略にする方向です。また、中国風の親しみやすさは持たせつつ、特定の既存作品に似せないことを創作方針とします。

タイトルの「棋子餅」は菓子の名前から採ったもので、作品全体の共通項か主人公側の名前かは未決定です。「王朝」にも今のところ歴史的な意味はありません。どちらも後から世界観を育てるための種として、複数案を企画書に残しています。

実在の中国を舞台にせず、中国語の一般名詞を組み合わせた [企画/地名案.md](企画/地名案.md) を作成しました。最初の地方候補は、霧と棚田の山地「雾阶盆地」です。村から山道、崖の桟道、関所、山寨、大河の渡しを経て城郭都市へ至る十二の地名候補があります。すべて未確定です。

流派は「色＋動物＋飯店などの店種」で命名し、大ボス、幹部、一般兵にも共通の色や動物を持たせる案があります。[企画/流派と組織名案.md](企画/流派と組織名案.md) に、紅熊亭、碧蛇茶楼、白鶴飯荘、黒牛麺館、金猿酒家、紫燕餃子房の組織案をまとめました。いずれも正式採用前の候補です。
