# Qizibing Dynasty ゲーム構想

更新日：2026年7月21日

## タイトル

- 英字表記：Qizibing Dynasty
- 読み：キージービン・ダイナスティー
- 漢字表記：棋子餅王朝

### タイトルの由来と未決定事項

「棋子餅」は中国語圏にある菓子の名前から採ったものです。現時点では、棋子餅が登場人物全体を結ぶ共通項なのか、主人公側の流派・道場・土地などの名前なのかは決まっていません。

「王朝」は中国風の雰囲気を出すために付けた言葉で、特定の歴史上の王朝や年代を意味しません。ゲーム内に本物の王朝が存在することや、王位を巡る物語も、タイトルだけを理由に確定させないものとします。

タイトルは先に生まれた言葉であり、今後の世界観を膨らませるための種として扱います。

### 棋子餅を世界観へ生かす案

以下は候補であり、まだ設定として確定しません。

- 各地の飯店で少しずつ形と味が違う、旅人の携帯食にする
- 道場の創始者が門弟へ分け与えた菓子に由来して、主人公側の紋章にする
- 円盤状または駒状の菓子を領地や人材になぞらえた、民間の盤遊びを作る
- 山道の宿場を結ぶ名物として、棋子餅の流通が交易路の広がりを表すようにする
- 天下の流派が集まる武術大会の記念菓子にして、物語の導入に使う
- 「王朝」は後世の人々がこの群雄割拠の時代へ付けた、少し大げさな呼び名にする

食べ物を世界の中心的な謎や神器にする必要はありません。背景の生活感、小さな笑い、主人公側の象徴、またはゲームシステム上の物資など、作品に合う大きさで採用します。

## ゲームの概要

古代中国風の世界を舞台にした、見下ろし型クォータービューの2Dストラテジーゲームです。

プレイヤーたちはそれぞれ武術の流派を持ち、道場を経営します。各地から個性的なユニットを集めて戦力を整え、各拠点を守る猛者を倒し、最終的にすべての拠点を制圧することを目指します。

時代考証には強く縛られず、古代中国らしい雰囲気と、愉快で親しみやすい創作世界を優先します。

## 画面と表現

- 基準画面サイズ：1920×1080（フルHD、16:9）
- 表現方式：2D
- 視点：見下ろし型クォータービュー
- 平面上に海、山、森、建物などのオブジェクトを配置する
- 山や丘などに高低差を付けられる構造を検討する
- 緑や山の間を、人の足で踏み固められた土の道が伸びる景観にする

### 高低差の案

見た目は2Dのまま、地形に段階的な高さ情報を持たせる2.5D方式が候補です。高低差は景観だけでなく、移動速度、見晴らし、射程、防御などへ影響させる余地があります。

## 世界観

- 古代中国風の山河、集落、城郭、道場、飯店がある
- 厳密な歴史再現ではなく、武侠、料理、動物などを自由に混ぜる
- 軍事一辺倒ではなく、どこか愉快で賑やかな雰囲気を持たせる

### 地理と道

- 海を主な舞台にはせず、山岳地帯と大河を世界の中心にする
- 山は移動を阻む地形であると同時に、天然の要塞として扱う
- 集落、城郭、道場などの拠点は山中に点在する
- 拠点同士を結ぶ長い土の道が、山林、峠、谷間を縫って続く
- 崖の中腹や断崖沿いにも細い道を通し、危険だが重要な経路にする
- 大河は交通、補給、防衛を左右する大きな境界として扱う余地がある

平地を塗り分けるだけの勢力争いではなく、「どの峠と崖道を押さえれば山中の拠点へ到達できるか」が分かる地形を目指します。道そのものが戦略上の主役になる構成です。

### 創作上の方向性

日本で広く親しまれている中国風の古典、物語、漫画、アニメ、ゲームなどから、親しみやすい雰囲気の存在は意識します。ただし、特定作品の人物、衣装、技、固有名詞、筋書き、画面構成に寄せず、棋子餅王朝独自の組み合わせとゲーム上の役割を作ります。

武術、道場、点心、山岳、大河、動物といった広い題材を、流派経営と道路・拠点制圧の仕組みに結び付けることで独自性を出します。西遊記を含む中国古典も発想源の一つになり得ますが、そのままの登場人物や旅程を再現することは企画の中心にしません。

## プレイヤーと勝利目標

1. プレイヤーは自分の流派と道場を持つ
2. 道場を経営しながらユニットを集め、育成または編成する
3. 各地の拠点へ進出し、そこを守る猛者に挑む
4. 猛者を倒して拠点を制圧する
5. すべての拠点を制圧して天下統一を果たす

複数の「プレイヤーたち」が同じ世界で競争または対戦する想定ですが、プレイ人数や、人間・コンピューターの内訳は未決定です。

## ユニットの例

- 少林寺拳法の僧侶
- 点心を売る飯店の娘々（ニャンニャン）
- 笹を食べているパンダ

武人だけでなく、商売人や動物も戦力または特殊能力を持つユニットになり得ます。流派との相性、雇用条件、役割、成長方法などは今後決めます。

## 今後決めたいこと

- 一人用、対戦用、協力用のどれを中心にするか
- リアルタイム制かターン制か
- マップを正方形グリッド、六角形グリッド、グリッドなしのどれにするか
- 道場経営で扱う資源と施設
- ユニットの入手、育成、戦闘、敗北時の扱い
- 流派ごとの個性と技の体系
- 拠点制圧戦のルール
- 高低差が移動や戦闘に与える効果
- 道が最初から存在するか、往来によって形成・成長するか
- 棋子餅を世界共通の文化、主人公側の象徴、単なる名物のどれにするか
- 「王朝」を劇中の国家、時代の通称、タイトルだけの言葉のどれにするか
