# 流派と組織名案

更新日：2026年7月21日

## 命名方針

流派は武術団体だけでなく、飯店、茶楼、酒家などを表向きの商売に持つ組織とします。名前は原則として次の三要素を組み合わせます。

> 色 ＋ 動物 ＋ 店または組織を表す言葉

例：**紅＋熊＋亭＝紅熊亭**

同じ組織の人物には色か動物の字を繰り返し使い、初見でも所属が分かるようにします。

- 組織名：色＋動物＋店種
- 大ボス：組織の象徴＋店主らしい称号＋個人名
- 幹部：料理や店内の役目を表す、組織固有の階級名
- 一般兵：色または動物＋身体部位や仕事＋「衆」「丁」など
- 非戦闘員：店員として自然な名前

個人名だけは所属名を外しても人物として覚えられるようにし、画面上では「称号・個人名」の形で表示します。

## 流派候補

### 紅熊亭

赤い提灯と巨大な鉄鍋を掲げる山の飯店兼道場。火力、重装、正面突破を得意とします。客へ腹いっぱい食べさせることを誇りとし、敵にも勝負の前には一杯出します。

- 大ボス：**紅熊大掌櫃・洪鉄山（こう・てつざん）**
- 幹部階級：**三大鍋**
  - 爆鍋頭・焦岩（しょう・がん）：火力と炒め技
  - 蒸鍋頭・桃蓮（とう・れん）：蒸気と目くらまし
  - 煮鍋頭・白湯（はく・とう）：回復と持久戦
- 上級兵：**紅熊鍋兵**
- 一般兵：**紅熊火丁**
- 補給員：**紅熊飯担ぎ**

「紅熊」の二字が、看板、衣服、兵種名のすべてに現れるため、最も分かりやすい基準流派になります。

### 碧蛇茶楼

霧深い谷で薬茶を扱う茶楼兼道場。毒、薬、偵察、偽情報を得意とします。悪の組織とは限らず、薬効を知らずに山へ入る者を信用していないだけかもしれません。

- 大ボス：**碧蛇楼主・柳翠蘭（りゅう・すいらん）**
- 幹部階級：**三茶頭**
  - 苦茶頭・沈葉（しん・よう）：弱体化
  - 香茶頭・蘭露（らん・ろ）：誘導と交渉
  - 冷茶頭・薄荷（はっか）：速度低下と治療
- 上級兵：**碧蛇茶衛**
- 一般兵：**碧鱗衆**
- 偵察員：**青舌の茶童**

### 白鶴飯荘

崖上の宿と食堂を営む、身軽な流派。回避、治療、高所移動を得意とします。争いを避けるように見えますが、山道と空模様を誰よりも掌握しています。

- 大ボス：**白鶴女将・鶴素月（かく・そげつ）**
- 幹部階級：**三羽膳**
  - 朝羽膳・明粥（めい・しゅく）：先行と索敵
  - 風羽膳・軽燕（けい・えん）：高所移動
  - 雪羽膳・清湯（せい・とう）：治療と浄化
- 上級兵：**白鶴羽衛**
- 一般兵：**白羽衆**
- 救護員：**白鶴粥番**

### 黒牛麺館

荷運び人や石工が集まる麺館兼道場。防御、建設、道路整備、補給を得意とします。動きは遅いものの、いったん道をつなぐと容易には退きません。

- 大ボス：**黒牛館主・牛岳（ぎゅう・がく）**
- 幹部階級：**三麺師**
  - 太麺師・石臼（せき・きゅう）：防御陣
  - 細麺師・糸雨（し・う）：拘束技
  - 刀削師・断壁（だん・ぺき）：障害物破壊
- 上級兵：**黒牛角衛**
- 一般兵：**黒角衆**
- 工兵：**黒牛道直し**

### 金猿酒家

行商人、旅芸人、酒造りが出入りする賑やかな酒家兼道場。奇襲、略奪、値切り、素早い撤退を得意とします。陽気で信用ならないように見えて、仲間同士の貸し借りは決して忘れません。

- 大ボス：**金猿酒仙・侯快天（こう・かいてん）**
- 幹部階級：**三杯将**
  - 一杯将・笑口（しょう・こう）：挑発
  - 二杯将・酔歩（すい・ほ）：回避
  - 三杯将・空壺（くう・こ）：物資奪取
- 上級兵：**金猿杯衛**
- 一般兵：**金尾衆**
- 工作員：**金猿酌童**

### 紫燕餃子房

街道沿いに小さな支店を増やす餃子房兼道場。連携、伝令、包囲、店舗網を利用した情報伝達を得意とします。戦闘員より店員の方が多く、どの町にも味方がいることが最大の強みです。

- 大ボス：**紫燕房主・燕双花（えん・そうか）**
- 幹部階級：**三包頭**
  - 水包頭・江雫（こう・だ）：水路担当
  - 焼包頭・鉄皮（てっ・ぴ）：防衛担当
  - 蒸包頭・雲衣（うん・い）：伝令担当
- 上級兵：**紫燕翼衛**
- 一般兵：**紫翼衆**
- 伝令員：**燕包み**

## 画面で所属を伝える方法

名前だけに頼らず、次の情報も統一します。

- 組織の色を衣服、旗、屋根、体力表示に使う
- 動物の印を看板、腰札、荷車へ付ける
- 大ボス、幹部、一般兵で同じ二字の組織名を表示する
- 飯店の料理と戦闘能力を対応させる
- 拠点名に「紅熊亭本店」「碧蛇茶楼・峡谷支店」のような看板名を付ける
