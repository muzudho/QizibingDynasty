# 四年周期と天の龍

更新日：2026年7月21日

## 基本構想

一回のゲームで流れる舞台内時間は、約四年間とします。

天下統一に百年を要する歴史物ではなく、各流派が同じ期限へ向かって一斉に動く、密度の高い四年間を描きます。四年という長さなら、無名の道場が勢力を広げる成長、季節の変化、ユニットの加入と離脱を表現しながら、ゲームプレイが際限なく長引くことを防げます。

## 四年の戦と四年の平和

この世界では、約四年間の戦乱と約四年間の平和が交互に訪れ、それが何世代にもわたって繰り返されてきたと考えられています。一巡は合計八年です。

1. 天巡龍が現れ、占い師たちが「天争の刻」を告げる
2. 各流派が王の座を求めて四年間争う
3. 次の天巡龍が現れ、勝者または停戦の時を告げる
4. 四年間の平和に入り、田畑、道路、店、道場、人口を立て直す
5. 再び天巡龍が現れ、次の戦乱が始まる

実際に超自然の力が戦争と平和を切り替えているとは限りません。あまりにも長く周期が続いたため、民衆、流派、商人、城主が暦に合わせて行動し、結果として予言どおりの歴史を繰り返している可能性があります。

平和期にも争いが完全になくなるわけではありませんが、天下を奪う大規模な進軍は強く忌避されます。流派は店と道場を増やし、弟子を育て、次の四年間へ備えます。ゲーム本編は、平和期の終わりから戦争期の四年間を主に扱います。

## 四年ごとの現象「天巡龍」

**天巡龍（てんじゅんりゅう）**は、四年に一度だけ空へ現れる、巨大な龍のように見える現象の仮称です。

大河から立ち上った湿気が山脈に沿って長大な雲となり、その内部を稲妻が頭から尾へ走ります。山々の上をうねる光と雲が、遠くからは一頭の龍が天下を巡っているように見えます。

人々は天巡龍を天の使いと考え、「龍が現れる日までに天下の道を結び、すべての拠点に認められた者が次の王に選ばれる」と信じています。

ただし、次のどれが真実かは断定しません。

- 天から遣わされた本物の龍である
- 雲、風、雷、大河、山脈が作る周期的な気象現象である
- 気象現象だが、人の運命を感じ取る不思議な力も持つ
- 王を選ぶという話だけが、流派を競わせるために作られた伝承である

画面には雲と光として描き、龍の鱗や顔をはっきり見せすぎないことで、見る者によって解釈が変わる存在にします。

## 占い師と自己成就する予言

占い師は天巡龍の雲の太さ、稲妻の向き、雷鳴の回数、風向きなどを読み、次の四年間を語ります。各地の占い師が同じ結論を出すとは限りませんが、有力な占い師の言葉は飯店、旅商人、寺院、道場を通じて急速に広がります。

- 「龍が東を向いた。東の流派に天命がある」
- 「尾が大河を打った。今年は渡し場を取る者が勝つ」
- 「雷が九度鳴った。九灶城から王が出る」

人々はその言葉を信じて兵糧を買い、城門を閉じ、流派へ入門し、道を修理します。占いが未来を当てたのか、皆が占いどおりに動いて未来を作ったのかは分かりません。

占い師は単なる背景人物ではなく、次の働きを持つユニットまたは中立人物にできます。

- 季節イベントや天候の予報を伝える
- 民衆の支持と流派の評判を変える
- 特定の道や拠点へ各勢力を誘導する
- 正しい観測、思い込み、買収された宣伝を混ぜる
- 同じ空を見ても異なる勝利条件を語る

## 科学と知識の水準

この世界には、天巡龍を気象現象として完全に説明できる科学はありません。一方で、人々が何も知らないわけではなく、長年の経験から季節、風、雲、大河の増水、作物の時期を細かく記録しています。

観測結果と宗教、迷信、商売上の都合が分離されていない段階です。優れた占い師は鋭い観察者でもあり、でたらめな占い師も偶然や人心操作によって予言を当てます。この曖昧さを世界の知識水準とします。

## ゲーム上の役割

天巡龍の出現日は、一局の明確な期限です。

1. ゲーム開始時に「龍が再び現れるまで、あと四年」と告げられる
2. プレイヤーと他流派は、道場経営、仲間集め、道路確保、拠点攻略を同時に進める
3. 季節が巡るたびに、山道、大河、収穫、交易、使える経路が変化する
4. 四年目が近づくと空模様が変わり、各勢力の争いが激しくなる
5. 天巡龍が現れるまでに全拠点を制圧し、天下から認められることを目指す

「全拠点制圧」と「四年の期限」を基本の勝利条件とします。期限時に最大勢力を自動的に勝者とするか、統一できなければ全員敗北または引き分けとするかは、ゲームモードを決める段階で確定します。

## 四年間の進行案

四季を一つの大きな区切りとすると、全体は **四年×四季＝十六季** になります。ターン制を採用する場合も、一季をそのまま一ターンにするとは限らず、一季の中に複数の行動期間を設けられます。

一年十二か月、一月三十日、七日一週を使わない独自暦の候補は [暦と時間表現案.md](暦と時間表現案.md) にまとめています。

### 一年目「芽吹」

- 小さな道場を構える
- 最初の仲間を集める
- 近隣の村、飯店、山道を知る
- 他流派の看板と噂が見え始める

### 二年目「競道」

- 峠、渡し場、宿場を巡って流派同士が衝突する
- 支店や道場を増やす
- 雪解け、長雨、増水などで進軍路が変わる

### 三年目「争覇」

- 大きな山寨や城郭都市を攻略する
- 敵流派の大ボスと本格的に戦う
- 同盟、吸収、裏切りが増える

### 四年目「龍待」

- 空に細長い雲が現れ、龍の前兆が始まる
- 残った流派が最後の拠点へ集中する
- 大河と山脈をまたぐ最終遠征を行う
- 天巡龍の出現とともに一局が決着する

## 季節と地形

- 春：雪解けで崖道が崩れやすいが、薬草と新人が増える
- 夏：大河が増水し、渡し場が減る。竹と作物が育つ
- 秋：食料と交易が豊かになり、遠征に向く
- 冬：峠が閉ざされる一方、凍結した浅瀬や隠し道を使える

季節は単なる見た目の変更ではなく、「今季はどの道を使うか」を考えさせる仕組みにします。

## タイトルとの接続候補

天巡龍が王を選ぶという伝承が定着しているなら、「王朝」という題にも劇中の意味を与えられます。ただし、王朝という言葉は雰囲気から付けたタイトルなので、この接続はまだ候補です。

棋子餅についても、天巡龍の日に焼く祝い菓子、四年前の備蓄食、または龍とは無関係な日常の食べ物のままにできます。
